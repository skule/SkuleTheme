SkuleTheme
============

Wordpress child theme for Skule hosted Wordpress websites, based on Wordpress' twentyten theme. Modifications include: 

  *  default Skule banner
  *  "supported by EngSoc" credit

Prerequisites
------------
You should know:

  *   PHP
  *   Wordpress (+ Child Themes)
  *   Buddypress
  *   HTML
  *   CSS
